# NavigationSplitViewColumn

A view that represents a column in a navigation split view.

```swift
struct NavigationSplitViewColumn
```

## Overview

A [NavigationSplitView](/documentation/swiftui/navigationsplitview) collapses into a single stack in some contexts, like on iPhone or Apple Watch. Use this type with the `preferredCompactColumn` parameter to control which column of the navigation split view appears on top of the collapsed stack.
