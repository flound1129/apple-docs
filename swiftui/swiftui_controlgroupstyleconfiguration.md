# ControlGroupStyleConfiguration

The properties of a control group.

```swift
struct ControlGroupStyleConfiguration
```

### Configuring the label

- [label](/documentation/swiftui/controlgroupstyleconfiguration/label-swift.property): A view that provides the optional label of the [ControlGroup](/documentation/swiftui/controlgroup).
- [ControlGroupStyleConfiguration.Label](/documentation/swiftui/controlgroupstyleconfiguration/label-swift.struct): A type-erased label of a [ControlGroup](/documentation/swiftui/controlgroup).

### Configuring the content

- [content](/documentation/swiftui/controlgroupstyleconfiguration/content-swift.property): A view that represents the content of the `ControlGroup`.
- [ControlGroupStyleConfiguration.Content](/documentation/swiftui/controlgroupstyleconfiguration/content-swift.struct): A type-erased content of a `ControlGroup`.
