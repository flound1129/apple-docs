# WindowProxy

The proxy for an open window in the app.

```swift
struct WindowProxy
```

### Instance Properties

- [id](/documentation/swiftui/windowproxy/id): The ID for the window, if one was provided.
- [phase](/documentation/swiftui/windowproxy/phase): The window’s current [ScenePhase](/documentation/swiftui/scenephase).
