# DefaultToolbarItem

A toolbar item that represents a system component.

```swift
struct DefaultToolbarItem
```

## Overview

Place this item in your toolbar to control where the system-provided item, like search, will be positioned.

### Initializers

- [init(kind:placement:)](/documentation/swiftui/defaulttoolbaritem/init(kind:placement:)): Creates a system-defined toolbar item from a `ToolbarDefaultItemKind` at the given `placement`.
