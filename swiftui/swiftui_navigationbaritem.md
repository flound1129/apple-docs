# NavigationBarItem

A configuration for a navigation bar that represents a view at the top of a navigation stack.

```swift
struct NavigationBarItem
```

## Overview

Use one of the [NavigationBarItem.TitleDisplayMode](/documentation/swiftui/navigationbaritem/titledisplaymode) values to configure a navigation bar title’s display mode with the [navigationBarTitleDisplayMode(_:)](/documentation/swiftui/view/navigationbartitledisplaymode(_:)) view modifier.

### Setting a title display mode

- [NavigationBarItem.TitleDisplayMode](/documentation/swiftui/navigationbaritem/titledisplaymode): A style for displaying the title of a navigation bar.

## See Also

### Configuring the navigation bar

- [navigationBarBackButtonHidden(_:)](/documentation/swiftui/view/navigationbarbackbuttonhidden(_:)): Hides the navigation bar back button for the view.
- [navigationBarTitleDisplayMode(_:)](/documentation/swiftui/view/navigationbartitledisplaymode(_:)): Configures the title display mode for this view.
