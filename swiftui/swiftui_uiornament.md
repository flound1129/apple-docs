# UIOrnament

The abstract base class that represents an ornament.

```swift
class UIOrnament
```

## Overview

You don’t create an instance of this class directly. Instead use [UIHostingOrnament](/documentation/swiftui/uihostingornament) when you want to add ornaments to a UIKit view controller.

## See Also

### Hosting an ornament in UIKit

- [UIHostingOrnament](/documentation/swiftui/uihostingornament): A model that represents an ornament suitable for being hosted in UIKit.
