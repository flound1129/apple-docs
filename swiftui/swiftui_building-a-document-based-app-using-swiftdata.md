# Building a document-based app using SwiftData

Code along with the WWDC presenter to transform an app with SwiftData.

## Overview

Learn how to use [Query](/documentation/SwiftData/Query), [Bindable](/documentation/SwiftUI/Bindable), [modelContainer(_:)](/documentation/SwiftUI/View/modelContainer(_:)), the [modelContext](/documentation/SwiftUI/EnvironmentValues/modelContext) environment variable, and [DocumentGroup](/documentation/SwiftUI/DocumentGroup) to integrate with the [SwiftData](/documentation/SwiftData) framework.

> **Note:**
> This sample code project is associated with WWDC23 session 10154: [Build an app with SwiftData](https://developer.apple.com/wwdc23/10154/).
>
