# AtomicRepresentable Implementations

### Type Methods

- [decodeAtomicRepresentation(_:)](/documentation/swift/int/decodeatomicrepresentation(_:)): Recovers the logical atomic type `Self` by destroying some `AtomicRepresentation` storage instance returned from an atomic operation.
- [encodeAtomicRepresentation(_:)](/documentation/swift/int/encodeatomicrepresentation(_:)): Destroys a value of `Self` and prepares an `AtomicRepresentation` storage type to be used for atomic operations.
