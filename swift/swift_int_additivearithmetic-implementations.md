# AdditiveArithmetic Implementations

### Operators

- [+(_:)](/documentation/swift/int/+(_:)): Returns the given number unchanged.
- [+=(_:_:)](/documentation/swift/int/+=(_:_:)-5g9ip): Adds two values and stores the result in the left-hand-side variable.
- [-=(_:_:)](/documentation/swift/int/-=(_:_:)-2aqsn): Subtracts the second value from the first and stores the difference in the left-hand-side variable.

### Type Properties

- [zero](/documentation/swift/int/zero): The zero value.
