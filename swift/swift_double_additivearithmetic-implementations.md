# AdditiveArithmetic Implementations

### Operators

- [+(_:)](/documentation/swift/double/+(_:)): Returns the given number unchanged.
- [+=(_:_:)](/documentation/swift/double/+=(_:_:)-5g9j0): Adds two values and stores the result in the left-hand-side variable.
- [-=(_:_:)](/documentation/swift/double/-=(_:_:)-2aqsq): Subtracts the second value from the first and stores the difference in the left-hand-side variable.

- [zero](/documentation/swift/double/zero): The zero value.
